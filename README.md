# MuralDesk

MuralDesk is a desktop app that lets you pin images, videos, links, GIFs, and notes onto your desktop like a visual sticky board.

It is built for people who want a second-screen inspiration wall, ambient video board, reference board, or creative workspace without keeping browser tabs open.

## Core Features

- Add local images
- Add local videos
- Add web links
- Loop videos
- Resize and move items
- Save layouts
- Multi-monitor friendly
- Desktop-style visual board

## Buildathon Goal

Build a working full desktop MVP in 24 hours.

## Tech Stack

- Replit
- React / Vite
- Electron or Tauri
- Local storage / file persistence
