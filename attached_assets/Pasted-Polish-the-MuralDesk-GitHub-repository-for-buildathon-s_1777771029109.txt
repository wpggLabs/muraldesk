Use current product truth:

MuralDesk is a local-first desktop mural layer for pinning images, videos, smart embeds, links, and notes over your workspace.

Current features:
- Electron desktop app
- Transparent frameless overlay
- Click-through empty areas
- Full-display Desktop Mode
- Current-display and all-display overlay modes
- Move window to any monitor, then expand over that display
- Drag/resize/lock/duplicate/delete cards
- Hover-only item controls
- Compact toolbar with More menu
- Tidy board action
- Focus mode if present
- Snap / board opacity / theme / accent controls if present
- Local images and videos
- Notes
- Normal web links
- Smart embeds:
  - YouTube
  - Vimeo
  - SoundCloud
  - Spotify
  - CodePen
  - direct video URLs
  - direct image URLs
- YouTube/embed Interact mode
- Export/import
- Portable backup if implemented
- Local-first storage:
  - localStorage for layout/metadata
  - IndexedDB for uploaded media blobs
- PWA/web version
- Vastago brand font loaded locally
- No login
- No backend
- No cloud storage

Important honest limitations:
- Not a full wallpaper engine yet.
- Not true OS wallpaper-layer integration yet.
- Multi-monitor support exists as current-display/all-display overlay, but not separate independent boards per monitor yet.
- Windows app is unsigned, so SmartScreen may warn.
- Electron desktop mode must be tested on a real desktop OS, not Replit.
- Replit cannot launch Electron because its container lacks GUI/GTK/Chromium system libraries.
